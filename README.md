# 🌌 黑暗森林模拟器 · Dark Forest Simulator

基于刘慈欣《三体》黑暗森林理论构建的交互式 3D 宇宙文明演化模拟器。

---

## 理论基础

> 宇宙就是一座黑暗森林，每个文明都是带枪的猎人，像幽灵般潜行于林间……他必须小心，因为林中到处都有与他一样潜行的猎人。如果他发现了别的生命，能做的只有一件事：开枪消灭之。

**两条基本公理：**
1. 生存是文明的第一需要
2. 文明不断增长和扩张，但宇宙中的物质总量保持不变

**核心概念：猜疑链 · 技术爆炸 · 黑暗森林打击 · 光粒 · 二向箔 · 黑域 · 咒语**

---

## 功能特性

### 宇宙模拟
- **600 星系 + 45 颗真实恒星**（太阳、比邻星、天狼星、TRAPPIST-1 等，含真实天文数据）
- **开普勒椭圆轨道**：行星沿真实轨道运行（离心率、倾角、升交点经度）
- **恒星生命周期**：主序星 → 红巨星 → 超新星爆发
- **150 个文明**分四代错峰诞生（古老/长者/成熟/年轻）
- **3D 自由飞行相机**：指针锁定 + WASD 操控 + 射线瞄准

### 文明机制
- **多星系殖民**：文明可扩张至多个恒星系，光粒打击只摧毁单个殖民地
- **科技树**：7 大领域（探测/隐匿/武器/推进/通讯/认知/经济）
- **技术爆炸**：压力或观测威胁触发非线性科技跃升
- **内部政治**：5 种政体 + 派系动态 + 恐惧-稳定度反馈
- **策略人格**：攻击性/谨慎度/合作性/扩张主义四维向量

### 打击与防御
| 手段 | 门槛 | 特点 |
|------|------|------|
| **光粒打击** ☀️ | 武器 ≥ 30 | 标准黑暗森林打击，摧毁恒星 |
| **二向箔** 📐 | 武器 ≥ 55 | 降维打击，永久改变宇宙结构 |
| **黑域** 🌑 | 认知+隐匿 ≥ 25 | 安全声明——自囚于低光速区 |
| **咒语** 🪄 | 完成追踪锁定 | 广播目标坐标，借刀杀人 |

### 外交与冲突
- **星际战争**：掌控范围重叠 + 技术相近 → 持久军事冲突
- **联盟外交**：低猜疑文明可结盟，科技共享，但有背刺风险
- **探测器**：发射无人深空探测器扩展视野
- **逃亡者**：打击下有 15% 概率出现幸存者舰队

### 宇宙灾难
- 伽马射线暴、超新星冲击波随机扫过宇宙
- 恒星老化膨胀吞噬近轨行星
- 星系边缘环境恶劣（宜居度惩罚）

## 操作指南

| 操作 | 功能 |
|------|------|
| **点击画布** | 进入飞行模式（指针锁定） |
| **移动鼠标** | 旋转视角 |
| **WASD** | 前后左右飞行 |
| **QE** | 升降 |
| **右键拖拽** | 平移 |
| **滚轮** | 缩放（向光标方向） |
| **↑↓←→** | 慢速旋转 |
| **空格** | 暂停/继续 |
| **→ 单步** | 逐 tick 前进 |
| **单击十字准星** | 锁定目标（显示轨道 + 详情面板） |
| **右键** | 取消锁定 |
| **ESC** | 释放指针 / 关闭弹窗 |

### UI 面板
- **右侧**：实时统计 + 文明数量折线图 + 灭绝原因饼图 + 事件日志
- **左上角**：锁定目标详情面板（单击准星触发）
- **底部**：速度控制 + 参数滑块 + 预设场景 + 📊总结 + 📥导出

## 项目结构

```
src/
├── main.ts            # 入口，模拟循环 + 渲染循环
├── config.ts          # 所有可调参数集中管理
├── types.ts           # 类型定义（文明/星系/行星/联盟/探测器…）
├── universe.ts        # 3D 宇宙空间 + 资源场
├── galaxy.ts          # 星系生成器 + 开普勒轨道计算
├── star_catalog.ts    # 真实恒星目录（~45 颗，含太阳系）
├── civilization.ts    # 文明实体（殖民/迁徙/增长/科技）
├── tech_tree.ts       # 科技树 + 技术爆炸
├── politics.ts        # 内部政治 + 派系动态
├── decision.ts        # 决策引擎（猜疑链/博弈/打击/咒语）
├── alliances.ts       # 联盟外交 + 科技传播
├── probes.ts          # 探测器系统
├── events_ext.ts      # 恒星生命周期 + 文明分裂 + 宇宙灾难
├── renderer.ts        # 3D Canvas 渲染器（自由相机/射线/十字光标）
├── charts.ts          # Chart.js 统计图表
├── ui.ts              # UI 控制面板
└── event_bus.ts       # 事件总线
```

## 技术栈

- **TypeScript** + **Vite** — 类型安全 + 极速 HMR
- **Canvas 2D** — 自定义 3D 透视投影渲染
- **Chart.js** — 统计图表
- **原生 HTML/CSS** — 无框架依赖
